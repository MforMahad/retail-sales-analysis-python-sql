# Online Retail Sales Analysis (Python & SQL)

## Overview
This project analyzes an online retail dataset to uncover sales trends, customer behavior, and revenue insights using Python (pandas) and SQL (MySQL).

## Dataset
- Source: Public online retail dataset
- Records: ~500,000 transactions
- Fields: InvoiceNo, StockCode, Quantity, InvoiceDate, UnitPrice, CustomerID, Country

## Data Cleaning
- Removed missing CustomerID values
- Converted InvoiceDate to datetime format
- Handled duplicates and invalid entries
- Created revenue feature (Quantity Ã— UnitPrice)

## Analysis Performed
- Revenue by country
- Monthly revenue trends
- Top spending customers
- Customer segmentation (High / Medium / Low value)

## Tools & Technologies
- Python (pandas)
- SQL (MySQL)
- Jupyter Notebook
- GitHub

## Key Insights
- The United Kingdom generated the highest revenue
- Sales peaked in November and declined in December
- A small group of high-value customers contributed significantly to total revenue
